# Monster Mechanics

## Reading a Stat Block

```
Name
Size, Type, Alignment

Armor Class: X (armor type)
Hit Points: Xd# + # (average)
Speed: X ft., [fly/swim/climb/burrow X ft.]

STR  DEX  CON  INT  WIS  CHA
X(±) X(±) X(±) X(±) X(±) X(±)

Saving Throws: (listed only if proficient)
Skills: (listed only if proficient)
Damage Immunities / Resistances / Vulnerabilities:
Condition Immunities:
Senses: darkvision X ft., passive Perception X
Languages: ...
Challenge: X (XP)

TRAITS (always active)
ACTIONS (taken on monster's turn)
BONUS ACTIONS (if any)
REACTIONS (if any)
LEGENDARY ACTIONS (boss monsters only)
LAIR ACTIONS (optional; boss monsters in their lair)
```

## Legendary Creatures

**Legendary Actions:** Can be taken at the end of *another creature's* turn, spending from a pool (usually 3). Resets at the start of the legendary creature's turn.

**Legendary Resistance (3/Day):** When the creature fails a saving throw, it can choose to succeed instead.

**Lair Actions (Initiative Count 20):** Boss monsters in their lair can take lair actions on initiative count 20 (losing all ties). Typically 3 options; only one can be used per round.

## Monster Saves & Skills

Monsters only list saves and skills they are **proficient** in. Proficiency bonus is determined by CR:

| CR | Proficiency Bonus |
|----|-------------------|
| 0–4 | +2 |
| 5–8 | +3 |
| 9–12 | +4 |
| 13–16 | +5 |
| 17–20 | +6 |
| 21–24 | +7 |
| 25–28 | +8 |
| 29–30 | +9 |

## Common Monster Traits (Glossary)

| Trait | Effect |
|-------|--------|
| **Ambusher** | Advantage on attacks in first round if it surprised the target |
| **Amphibious** | Can breathe air and water |
| **Flyby** | Doesn't provoke opportunity attacks when it flies out of reach |
| **Keen Senses** | Advantage on Perception checks using specified sense |
| **Magic Resistance** | Advantage on saving throws against spells and magical effects |
| **Magic Weapons** | Weapon attacks count as magical for overcoming resistance |
| **Pack Tactics** | Advantage on attacks if ally is adjacent to target (not incapacitated) |
| **Regeneration** | Regains HP at start of turn unless a specific damage type was dealt |
| **Spellcasting** | Can cast spells; uses listed ability and save DC |
| **Spider Climb** | Can climb on difficult surfaces including ceilings without check |
| **Sunlight Sensitivity** | Disadvantage on attacks and Perception checks in direct sunlight |

## Multiattack

Most monsters with a Multiattack action make the specified number and type of attacks in one action. These don't trigger the two-weapon fighting rule.

---

## Monster Spellcasting

Monsters using the **Spellcasting** trait have:
- A spellcasting ability (INT, WIS, or CHA)
- A spell save DC and spell attack bonus
- A list of spells by frequency (At will / 3/day each / 1/day each / etc.)

**Innate Spellcasting** works the same but typically requires no material components.

---

## Running Monsters Effectively

- **Morale:** Consider having monsters flee or surrender below ~30% HP. Not all monsters fight to the death.
- **Tactics:** Intelligent monsters use cover, target weaker party members, and coordinate. Beasts act instinctually.
- **Villain Retreats:** Allow a recurring villain to escape when defeated — don't let early combat kills ruin a great antagonist.
- **HP as a Range:** You can treat monster HP as a range ±10–20% for dramatic pacing.
- **Adjusting on the fly:** An "at-will" ability can be treated as limited if combat is going badly; DM's prerogative.
