# Anchor NPCs — The Badger's End

Anchor NPCs are permanent fixtures. Every DM plays them consistently. When in doubt, keep them true to their profile rather than inventing new traits — save invention for the randomized NPCs.

**The golden rule:** Anchor NPCs can grow and change, but only through things that happen at the table. Update the session notes after every session they appear in.

## Promoting a Randomized NPC to Anchor

When the group clearly loves a randomized NPC — they ask about them, they look for them, they remember their name — promote them:

1. Move their entry from the world-state log into this folder
2. Fill in their profile using what happened at the table
3. Mark them as Anchor status in the world-state doc
4. Add them to the table above

---

## Filling in Placeholders

When a DM introduces a placeholder NPC for the first time:
- Give them a name and note it in their file and the world-state doc
- Choose a race that fits the moment
- Play to their profile — personality is more important than details
- Add a session note afterward with what landed

Placeholder details should be decided at the table, not alone in prep. Let the game shape them.
